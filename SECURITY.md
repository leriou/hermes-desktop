# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it **privately**:

- Open a GitHub [security advisory](https://github.com/fathah/hermes-caduceus/security/advisories/new)

**Do not open a public issue for security vulnerabilities.**

## Scope

- The desktop application (this repository)
- The Tauri IPC layer between the native shell and the renderer
- Credential and API key handling

## Out of Scope

- Vulnerabilities in the Hermes Agent CLI core (report to [hermes-agent](https://github.com/NousResearch/hermes-agent))
- Issues in third-party LLM provider APIs
- Social engineering or phishing attacks
